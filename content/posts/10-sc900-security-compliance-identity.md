Naast Azure heb ik ook SC-900 (Microsoft Security, Compliance, and Identity Fundamentals) op mijn lijst staan. Dit is de security-tegenhanger van AZ-900.

## Waarom SC-900?
Security is overal. SC-900 geeft een overzicht van:
- Microsoft security oplossingen
- Compliance mogelijkheden
- Identity management met Microsoft Entra ID

## Belangrijke onderwerpen

### Identity met Microsoft Entra ID
- **Entra ID** (voorheen Azure AD) — cloud identity service
- **Multi-Factor Authentication** — extra beveiligingslaag
- **Conditional Access** — voorwaardelijke toegang op basis van context
- **Privileged Identity Management** — just-in-time admin toegang

### Security oplossingen
- **Microsoft Defender for Cloud** — beveiliging van multicloud workloads
- **Microsoft Sentinel** — SIEM/SOAR voor threat detection
- **Microsoft Defender XDR** — unified security platform

### Compliance
- **Microsoft Purview** — data governance en compliance
- **Compliance Manager** — risico-inschatting en compliance score
- **Service Trust Portal** — Microsoft's compliance documentatie

## Mijn ervaring tot nu toe
Ik heb SC-900 gestart op Microsoft Learn. Het sluit mooi aan op AZ-900 omdat veel concepten overlappen, maar dan met security-perspectief. De shared responsibility komt terug, maar nu met focus op security controls.

Na SC-900 wil ik dieper in Microsoft Sentinel en Defender for Cloud duiken — praktische tools die direct relevant zijn voor een SOC analyst rol.
