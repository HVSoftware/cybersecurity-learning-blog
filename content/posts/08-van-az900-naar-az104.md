AZ-900 is het fundament. AZ-104 (Azure Administrator) is de volgende stap — een stuk diepgaander en praktischer.

## Wat is het verschil?

| Aspect | AZ-900 | AZ-104 |
|--------|--------|--------|
| Niveau | Beginner | Intermediate |
| Vragen | 30-45 minuten | 90-120 minuten |
| Type | Concepts, terminologie | Praktijk, configuratie |
| Labs | Geen | Ja (Azure Portal, CLI, PowerShell) |

## AZ-104 exam domeinen

1. **Identiteit beheren** (15-20%)
   - Microsoft Entra ID (voorheen Azure AD)
   - Gebruikers, groepen, rollen
   - Conditional Access

2. **Governance en compliance** (10-15%)
   - Azure Policy, RBAC, Management Groups
   - Tags, resource locks, budgets

3. **Storage beheren** (15-20%)
   - Storage accounts, blobs, files
   - Replicatie (LRS, GRS, ZRS)
   - Security (SAS tokens, firewall)

4. **Compute** (20-25%)
   - Virtual Machines, VM Scale Sets
   - App Services, Containers, AKS
   - Availability Sets en Zones

5. **Networking** (20-25%)
   - VNet, subnetten, peering
   - Load Balancer, Application Gateway
   - VPN Gateway, DNS
